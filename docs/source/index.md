## Overview

This workspace provides a complete development environment for:
- ESP32 firmware development with FreeRTOS
- micro-ROS integration for ESP32
- ROS 2 Humble host environment
- Docker-based containerization

## Features

- **Docker Compose**: Complete multi-container development setup
- **ROS 2 Humble**: Latest ROS 2 distribution
- **micro-ROS Agent**: UDP bridge for ESP32 communication
- **Sphinx Documentation**: This documentation system
- **VS Code Integration**: C++ IntelliSense and debugging support
